สร้าง web application สำหรับระบบขอเอกสารทางการศึกษาออนไลน์แบบง่ายๆ สำหรับมือใหม่
ระบบขอเอกสารทางการศึกษาออนไลน์สำหรับนักศึกษา
สร้างโค้ดตัวอย่างโดยใช้เทคโนโลยีพื้นฐานที่เหมาะกับมือใหม่:
เบื้องต้นสำหรับการสร้าง web application ระบบขอเอกสารทางการศึกษาออนไลน์ โดยไม่ลงรายละเอียดโค้ด
คำแนะนำสำหรับระบบขอเอกสารทางการศึกษาออนไลน์
1. องค์ประกอบพื้นฐานของระบบ
- ระบบลงทะเบียน/เข้าสู่ระบบ สำหรับนักศึกษา
- หน้ารายการเอกสาร แสดงประเภทเอกสารที่ขอได้ (transcript, หนังสือรับรอง ฯลฯ)
- แบบฟอร์มขอเอกสาร สำหรับกรอกรายละเอียดต่างๆ  แนบเอกสารยืนยันตัวตน
- ระบบติดตามสถานะ ให้นักศึกษาเช็คความคืบหน้า
- ส่วนของผู้ดูแลระบบ สำหรับจัดการคำขอ
2.Web Server:
- Ubuntu 22.0
- url: https://form.devapp.cc
3.Frontend:
-HTML, CSS, Bootstrap (ช่วยให้สร้าง UI สวยงามได้ง่าย)
-JavaScript พื้นฐาน (ไม่จำเป็นต้องใช้ framework ซับซ้อน)
-หน้าล็อกอิน
-หน้าแสดงรายการเอกสารให้เลือก
-แบบฟอร์มกรอกข้อมูล
-หน้าแสดงสถานะคำขอ
4.Backend:
-Node.js + Express: port 4700
-ระบบยืนยันตัวตน
-การจัดการข้อมูลคำขอ
-ระบบแจ้งเตือน (อีเมล/SMS)
